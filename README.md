## How to Use

1. **Set Up Budget Categories:**
   - Go to "Budget Setup" tab
   - Enter category name and budget amount
   - Click "Add Category"

2. **Track Spending:**
   - Go to "Track Spending" tab
   - Select a category
   - Enter amount spent and date
   - Click "Add Spending"

3. **View History:**
   - Go to "Spending History" tab
   - Use filters to view specific categories or time periods
   - See spending trends on the line chart

## Data Storage

- All data is stored locally in your browser using localStorage
- No data is sent to any server
- Data persists between sessions

## Browser Support

Works on all modern browsers:
- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+

## Installation

Simply download the files and open `index.html` in your browser. No server or installation required!

## Privacy

This app respects your privacy:
- No bank connections
- No data collection
- No third-party tracking
- All data stays on your device
